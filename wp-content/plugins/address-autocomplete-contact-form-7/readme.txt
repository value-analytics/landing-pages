Contact form 7 address autocomplete feature. We are using google maps api. https://maps.googleapis.com/maps/api

== Description ==

This is an addon for contact form 7 for address autocomplete functionality. Google place api is used for fetching addresses.

== Installation ==
1) Upload the plugin.
2) Activate the plugin.
3) Add google place api key in Google Place API section under contacts menu.
4) Add address autocomplete field to contact form. Don't forget to add an id "address_geo_autocomplete-xxx" to address field for autocomplete feature.
